# 微信小程序－省市区联动选择器

### 说明：

实现省市区三级联动选择。

### 数据接口:

- https://api.getweapp.com/vendor/zto/baseArea

### 目录结构：

- utils — 存放项目公共库文件
- selectarea — 存放级联组件文件
- pages — 存放项目页面文件

### 开发环境：

微信web开发者工具 v0.11.112301
